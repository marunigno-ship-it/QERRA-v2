# Contributing to QERRA-v2

QERRA-v2 is an MIT-licensed research project developing the first quantum-aware humanoid control stack with unbreakable ethical enforcement.

## Areas for collaboration
- Add new real-life ethical vectors in `/vectors` (JSON format)
- Improve the Qiskit ↔ ROS2 bridge in `QERRA.PY`
- Run and verify entanglement circuits on IBM hardware
- Extend documentation and translations

Fork → develop → open a Pull Request.

All contributions are credited.  
Real-life ethical scenarios only – no synthetic data.
